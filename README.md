# Vólund Clean Foundation

## Objetivo

Esta branch contém uma fundação técnica limpa para uma nova construção do site institucional da Vólund.

## Branches preservadas

- main
- rebuild/frontend-v2
- rebuild/frontend-v3-unified

## Elementos preservados

- `.claude`;
- nove skills;
- `reference-materials`;
- memória do Obsidian;
- referências visuais;
- materiais institucionais;
- assets oficiais;
- copy da v3;
- referência do preloader;
- referência do footer.

## Elementos removidos

- implementação visual da v1;
- implementação visual da v2;
- implementação visual da v3;
- hero com cases;
- canvas geométrico;
- cases temporários;
- arquitetura Matéria em Suspensão;
- documentação de QA e screenshots das versões rejeitadas;
- dependências visuais antigas.

## Próxima etapa

A próxima missão deve construir somente:

- fundação visual;
- header;
- preloader;
- hero;
- transição para a segunda seção;
- desktop;
- mobile.

Não deve construir o site inteiro.

## Regra

“Esta branch não é v4. É uma fundação limpa para uma nova construção.”

“As versões anteriores não são baseline visual. Elementos aprovados da v3 estão isolados em reference-materials/approved-from-v3.”
